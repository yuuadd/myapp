・検索機能の実装(仮)

例えば「ラーメン」と検索したら情報のどこかに「ラーメン」という文字列が入った投稿が一覧ででます。

「中島　ラーメン」といった、複数の条件を指定した検索も一応可能です。

(「中島　ラーメン　ディナー」みたいな三つ以上の条件はまだ試していません)



・投稿機能の修正

投稿機能で、「名前をいれたらサジェストがでてきて、それをクリックすると名前含むすべての情報が入力される」機能を実装しました。

現時点では「サイノ、やきとりの一平　學亭、竹よし　ラーメンハウス、徳寿　室蘭店、アルコット」だけ入力することが出来ます。

いちおう手書きで入力する機能は残してあります。

不便な点として、例えば「やきとり」といれた時は一平が候補としてでますが、「とくじゅ」と打っても、漢字じゃない(文字列が一致しない)のでサジェストにでません。
